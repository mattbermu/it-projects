# Project 03: Linux Virtual Machine Administration Basics

## Objective
Build foundational Linux system administration skills by installing and managing an Ubuntu virtual machine in a local VirtualBox environment.

## Environment
- VirtualBox
- Ubuntu Linux
- Terminal / Command Line Interface

## Tasks Performed
- Installed Ubuntu Linux on a virtual machine
- Navigated the Linux file system using terminal commands
- Created and managed files and directories
- Modified file permissions and ownership
- Created and managed local users and groups
- Configured sudo access and troubleshot permission issues
- Installed and managed software packages
- Monitored system resources and inspected network configuration

## Skills Demonstrated
- Linux system administration fundamentals
- User and group management
- File permissions and access control
- Command-line proficiency
- Troubleshooting permission and access issues
- Technical documentation

## Outcome
Successfully practiced and documented core Linux administration tasks commonly required in IT support and junior system administration roles.
