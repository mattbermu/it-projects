# Project 02: Help Desk Ticket Resolution Lab (Windows 11)

## Objective
Simulate real-world help desk scenarios by resolving common user account, network, and access issues in a Windows environment.

## Environment
- Windows 11
- Local administrator and standard user accounts
- Shared folders and mapped drives

## Tasks Performed
- User account enable/disable
- Password resets
- Network adapter troubleshooting
- Shared folder permissions
- Drive mapping
- Simulated Microsoft 365 access support

## Skills Demonstrated
- Help desk troubleshooting
- Windows user account management
- Networking fundamentals
- Access control and permissions
- Technical documentation

## Outcome
Successfully resolved multiple simulated help desk tickets and documented troubleshooting steps and resolutions.
